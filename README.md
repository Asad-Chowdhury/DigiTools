# 🚀 DigiTools

## 📌 Project Overview

**DigiTools** is a subscription-based SaaS platform designed to empower developers, marketers, and analytics professionals with advanced tracking, conversion optimization, and marketing automation tools.

It simplifies the setup and management of modern marketing infrastructure — from tag management to analytics integration — all within a clean and intuitive interface.

---

## 🎯 Purpose

DigiTools aims to centralize and streamline marketing technology workflows, making it easier to:

- Implement tracking systems
- Monitor conversions
- Automate marketing processes
- Manage tools efficiently in one place

---

## 👥 Target Users

- Marketing Teams
- Developers
- Analytics Specialists
- Growth-focused Businesses

---

## ⚙️ Features

### 🛒 Interactive Cart Management

- Add products to cart dynamically
- Display total item count in navbar
- Remove individual items
- Clear entire cart during checkout

### 📦 Dynamic Product Display

- Load products from custom JSON data
- Responsive 3-column layout
- Display pricing, tags, and tool features

### 🔔 Real-Time Notifications

- Instant alerts using **React-Toastify**
- Feedback on:
  - Add to cart
  - Remove item
  - Checkout actions
